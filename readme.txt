**Aplicação Lista de Recados - Back-end e Front-end**

Este é um projeto de exemplo que demonstra como criar uma aplicação Lista de Recados com um back-end em Node.js (Express) e um front-end simples em HTML, Bootstrap e JavaScript.

**Instruções de Inicialização**

**Back-end:**

1. Certifique-se de ter o Node.js instalado em sua máquina.

2. Crie um diretório para o projeto e coloque o arquivo `script.js` dentro dele.

3. No terminal, navegue até o diretório do projeto e execute o seguinte comando para instalar as dependências do back-end:

   ```terminal
   npm install express sqlite3 bcrypt jsonwebtoken

instalar servidor:
npm install -g http-server

iniciar servidor:
http-server
